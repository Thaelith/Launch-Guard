# Roadmap

LaunchGuard Lite v0.1 is the initial release focusing on essential read-only pre-launch checks.

## v0.2 (Planned)

- JSON export of reports
- Report history (last N runs saved to disk)
- Configurable check severity overrides

## v0.3 (Planned)

- Vault economy checks (verify economy provider is set up)
- PlaceholderAPI checks (verify configured placeholders resolve)
- Additional location safety blocks

## v0.4 (Planned)

- LuckPerms group audit (check group permissions against expected config)
- HTML reports with configurable template
- Startup checks (run automatically on server start)

## v0.5 (Planned)

- Discord webhook reports
- Scheduled checks (run on a timer)
- Launch Mode (guided pre-launch workflow)

## Future

- Multi-language message support
- API for third-party checks
- Web panel integration

## Not Planned

- Executing arbitrary commands configured by server owners
- Modifying server state (plugins, worlds, blocks, permissions)
- Teleporting players
- Sending analytics or telemetry
- Premium-only check categories (all checks are available to all users)
